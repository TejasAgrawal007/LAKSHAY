C Overview


What is C?⁣

  1. Since the late 19th century, C has been a popular programming language for general-purpose use.
  2. C language was developed by Dennis M. Ritchie at bell laboratory in early 1970s
  3. Its applications are very diverse. It ranges from developing operating systems to databases and all.⁣ It is system programming language used to do low level programming (e.g., driver or kernel)
  4. Even if it’s old, it is still a very popular programming language.⁣
  5. As the whole UNIX operating system was written in C, it has a strong association with the operating system⁣
  6. C has also been used widely while creating iOS and Android kernels.⁣
  7. MySQL database is written using C.⁣
  8. Ruby and Pearl are mostly written using C.⁣
  9. Embedded Systems are created using C⁣
